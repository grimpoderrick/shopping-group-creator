# Shopping Group Creator

Shopping Group Creator exists to help the project team turn raw shopping or purchase-level data into clean grouped summaries that are ready for analysis.

The app is for project team members who work with:
- product coding files
- respondent or person-level purchase data
- grouped outputs for analysis, reporting, or modeling

Users upload a product coding file and a shopping data file, then choose the product attribute they want to group by, such as category, subcategory, or brand. The app maps purchase variables to the coding file, aggregates units and dollars, and returns a downloadable summary file.

In short, this project saves the team time, reduces manual file wrangling, and creates a more consistent way to prepare shopping-group outputs across studies.
